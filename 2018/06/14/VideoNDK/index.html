函数指针的定义

  • int (*fptr)(int,int) // (*func)中括号是必要的，这会告诉编译器我们声明的是函数指针而不是声明一个具有返回型为指针的函数

  • 指针变量 : *fptr ,

  • 如果*fptr变为函数名，整个就是一个函数

函数指针的使用

fptr = func;

int x = fptr(5,8);
